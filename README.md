# NEAP Client-Side

This is the client-side of our NEAP App. It consumes data from the server. It's Angular.

## Development

1. Create a new component
1. Setup a service
1. Controller
  - Access the service
1. Service
  - AJAX
1. Add `access-control` header check to server
1. Controller
  - update the view

## Deployment

```
$ git subtree push --prefix src origin gh-pages
```
